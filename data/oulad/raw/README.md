# OULAD Raw Data

Place the seven official OULAD CSV files in this directory before running the experiments:

- `studentInfo.csv`
- `studentRegistration.csv`
- `studentAssessment.csv`
- `assessments.csv`
- `studentVle.csv`
- `vle.csv`
- `courses.csv`

Download source:

https://figshare.com/articles/dataset/OULAD_Open_University_Learning_Analytics_Dataset/5081998

Download `anonymisedData.zip`, extract it, and copy the CSV files here. The raw CSV files are ignored by git to keep the repository lightweight and to avoid redistributing dataset files inside the code repository.
